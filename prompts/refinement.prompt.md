# Refinement Prompt

Improve the current project to feel more premium, trustworthy, and production-ready.

Focus on:

- stronger visual hierarchy
- cleaner spacing rhythm
- better typography
- stronger CTA design
- more convincing proof
- improved mobile responsiveness
- subtle hover and motion states
- clearer section transitions

Constraints:

- preserve existing architecture
- keep styling consistent with the project
- avoid unrelated rewrites
- maintain lint/build success

When complete:

1. run the project verification script
2. summarize modified files
3. explain design decisions
4. identify remaining weaknesses honestly
